CSS Positioning: Quick Overview

Relative (position: relative;)

An element with position: relative is positioned based on its normal place in the document flow. It still occupies space in the layout, but can be moved using the top, right, bottom, or left properties. This is useful for making small adjustments without disrupting other elements on the page.

Absolute (position: absolute;)

Elements with position: absolute are removed from the document flow and positioned relative to their nearest positioned ancestor. If no such ancestor exists, they are positioned relative to the document. This method is ideal for placing elements like tooltips, popups, or dropdowns that need specific placement without affecting the surrounding layout.

Fixed (position: fixed;)

An element with position: fixed is also removed from the normal document flow but is positioned relative to the viewport, meaning it stays in place even when the page is scrolled. This is commonly used for elements that need to remain visible, such as sticky headers, footers, or floating navigation menus.
